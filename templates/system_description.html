Описание Функционала Системы

1. Управление Пользователями

Система предоставляет полный цикл управления учетными записями пользователей.

Регистрация
  • Создание новой учетной записи с указанием имени пользователя, email и пароля.
  • Защита от ботов с помощью капчи при регистрации.
  • Подтверждение email.
Аутентификация
  • Вход в систему по email и паролю.
  • Функция «Запомнить меня».
  • Восстановление пароля через email с генерацией токена и защитой капчей.
Профиль Пользователя
  • Просмотр и редактирование личной информации (имя, контактные данные, название компании).
  • Просмотр статистики: количество опубликованных объявлений, отзывы.
  • Смена пароля.
  • Просмотр публичного профиля других пользователей с их объявлениями и отзывами.

2. Управление Товарами

Основная сущность системы — товары и объявления.

Создание Объявлений
  • Добавление товара с указанием названия, описания, цены, состояния (новое/б/у).
  • Выбор категории товара (древовидная структура).
  • Указание местоположения (Регион, Город).
  • Загрузка нескольких изображений товара с возможностью предпросмотра.
  • Указание наличия доставки и включения НДС в цену.
Редактирование и Управление
  • Редактирование всех полей существующего объявления.
  • Смена статусов: Опубликовать, Снять с публикации (черновик), Готов к публикации.
  • Удаление объявления.
  • Автоматическое снятие с публикации по истечении срока (30 дней).
Просмотр
  • Детальная страница товара с галереей изображений.
  • Отображение продавца и его рейтинга.
  • Счетчик просмотров товара.
  • Жалоба на объявление.

3. Поиск и Навигация

Инструменты для быстрого поиска нужных товаров.

  • Категории: Иерархическая структура категорий с навигацией по "хлебным крошкам".
  • Поиск: Глобальный поиск по названию и описанию.
  • Фильтры: Фильтрация по категориям, сортировка по цене, дате, состоянию.

4. Личный Кабинет (Dashboard)

Единый центр управления для пользователя.

  • Мои Объявления: Список всех товаров с различными видами отображения (Плитка, Список, Таблица).
  • Статистика: Просмотры, оставшееся время публикации, статус товара.
  • Избранное: Отдельный раздел для сохраненных интересовавших товаров.

5. Система Отзывов и Рейтингов

Механизм доверия между участниками сделок.

  • Возможность оставить отзыв о продавце (оценка 1-5 и комментарий).
  • Расчет среднего рейтинга пользователя.
  • Публичный профиль с историей отзывов.

6. Административная Панель

Интерфейс для управления системой.

  • Управление категориями и географией (регионы/города).
  • Управление пользователями (блокировка, просмотр).
  • Мониторинг системы и обратная связь.
